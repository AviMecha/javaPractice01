JavaScript Variables

x, y, z는 선언되지 않습니다.

처음 사용할 때 자동으로 선언됩니다.





사용하기 전에 항상 변수를 선언하는 것이 좋은 프로그래밍 방법으로 간주됩니다.





var 키워드는 1995년부터 2015년까지 모든 자바스크립트 코드에서 사용되었습니다.

let와 const 키워드는 2015년에 자바스크립트에 추가되었습니다.

var 키워드는 이전 브라우저용으로 작성된 코드에서만 사용해야 합니다.

const를 사용할 수 없는 경우에만 let를 사용





let, const 변수 선언은 섞어서도 사용할 수 있음





문자열은 따옴표로 감싼다.

숫자는 따옴표로 감싸지 않는다.





하나의 문장에서 여러 변수를 선언할 수 있다.





값 없이 선언된 변수는 undefined로 나타남





자바스크립트 변수 재선언

var로 선언한 JavaScript 변수를 다시 선언해도 해당 값이 손실되지 않습니다.

변수 carName은 다음 문장이 실행된 후에도 "Volvo" 값을 유지합니다.





대수학과 마찬가지로 = 및 +와 같은 연산자를 사용하여 자바스크립트 변수로 산술을 수행할 수 있습니다:





따옴표가 들어가면 "5" + 2 + 3 문자열로 인식한다





달러 기호는 자바스크립트 이름의 문자로 처리된다





자바스크립트 밑줄(_)

JavaScript는 밑줄을 문자로 처리하므로

_를 포함하는 식별자는 유효한 변수 이름입니다
